ML Demo Notebook Examples - Llama v2 AR-128 Weightshared Model
------------------------------------------------------------
This package includes Jupyter Notebook examples, code, documentation and assets
to optimize, prepare, convert and execute the Llama v2 AR-128 Weight Shared
Generative AI (GenAI) model on Snapdragon-based target devices.

Please go to each of the following notebook example folders inside llama2-ar128 
folder in sequence and follow the instructions in the README files therein:

- example-1:
	- PyTorch Model optimization and export using AIMET

- example-2:
	- Preparation and conversion of ONNX model to Qualcomm NN format

- example-3:
	- Qualcomm NN Model Execution on Windows on Snapdragon (WoS) device
